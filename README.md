# Golf Shot Tracker

Static mobile-friendly golf shot tracker ready for GitHub Pages.

## What It Does

- Tracks tee club and tee result
- Tracks approach distance, club, and result
- Lets you rate strike quality from 1 to 5
- Saves entries in browser `localStorage`
- Supports install to a phone home screen

## Publish To GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Push to the `main` branch.
4. In GitHub, open `Settings` -> `Pages`.
5. Under `Build and deployment`, choose `GitHub Actions`.
6. The included workflow will publish the site automatically.

After deployment, your app will be available at:

`https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/`

## Install On Phone

### iPhone

1. Open the GitHub Pages URL in `Safari`.
2. Tap `Share`.
3. Tap `Add to Home Screen`.

### Android

1. Open the GitHub Pages URL in `Chrome`.
2. Tap the menu.
3. Tap `Install app` or `Add to Home screen`.

## Notes

- Data is stored locally in the browser on that phone.
- If you clear browser storage, the saved round data is removed.
